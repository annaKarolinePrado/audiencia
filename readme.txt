Para rodar o projeto em php:

1° Baixar a versão mais atual do xampp ( já vem com o php 7)
2° Acessar site do composer e baixar e instalar
3° Baixar o projeto no git
4° Acessa a pasta do projrtojeto e rodar os comandos  do email que irão:
 instalar o JWT, as dependencias do composer, os enuns e as exceptions
5° Iniciar o servidor do xampp
6° Acessar o phpmyadmin em http://localhost
7° Criar um banco de dados
8° No projeto copiar o arquivo .env exemplopara um arquivo chamdo .env e alterar o nome do banco de dados 
para o banco que foi criado.
9ºCriar o tabelas no banco
     php artisan migrate
10ºExecutar o projeto 
     php artisan serve
